Курсовая работа Буглака Сергея группа 80.

Введение.
В процессе выполнения данной работы я написал код по распознаванию лиц на фотографии
с использованием библиотеки openCV, а также разработал телеграм бота для парсинга погоды и 
курса валют с использованием библиотек telebot, requests, pyowm.
Мне хотелось разработать что-то полезное и наблюдая за своими коллегами я обратил внимание, 
что очень часто, по утрам, они обсуждают погоду и курсы валют. Поэтому и родилась такая идея
с телеграм ботом. А распознание лиц я выбрал т.к. хочу научиться обрабатывать видео и фото.

1. Распознавание лиц (файл 1.py).
Для начала мы импортируем библиотеку openCV. На GitHub мы находим уже готовую нейронную сеть 
для распознания лиц и создаем в проекте файл haar_face.xml. После этого читаем ранее подготовленное
изображение и уменьшаем его размер в 2 раза для комфортной работы. Переводим изображение из цветного
в черно-белое и с помощью файла haar_face.xml находим лица при помощи нейронной сети. Определяем
количество распознанных лиц. При помощи цикла пробегаемся по распознанным лицам и рисуем квадрат
по их координатам оппределенного цвета, величины и толщены граней. После этого  выводим результат 
распознавания лиц на цветное изображение.

2. Телеграм бот (файл bot_3.py).
Для начала импортируем необходимые библиотеки: telebot, requests, pyowm, types.
В Telegram заходим на botFather регистрируем нового бота и получаем токен. Вводим токен нашего бота
Используя библиотеку requests делаем запрос в Беларусбанк о курсах валют. Из полученного списка словарей
берем необходимые значения по ключам соответствующим выбранным валютам. Далее используя декоратор
@bot.message_handler создаем уже сам код бота. Сам код состоит из множества функций которые отвечают 
за отправку запросов, их обработку, получение информации и вывод ее в бот.

Заключение.
Написанные в результате курсовой работы коды не являются окончательными:
код для распознания лиц на фото будет использоваться для разработки кода по распознанию лиц на видео
и сравнение их с изображением на фото;
разработанный телеграм бот будет использован для разработки такого же бота, но с изменением, для 
ведомств быстрого реагирования.
